Welcome to NewsApi!

You may use it by making requests to following links *:
  • '/api/v1/auth' - [POST] 
  • '/api/v1/news/' - [GET, POST]
  • '/api/v1/news/<id>/' - [GET, PUT, PATCH, DELETE] **
  • '/api/v1/news/<id>/comments/' - [GET, POST]
  • '/api/v1/news/<id>/comments/<id>/' - [GET, DELETE] **
  • '/api/v1/news/like/' - [GET]
  • '/api/v1/news/unlike/' - [GET]

For detail information about project and requests examples follow link:

Project info

* All of the links mentioned above request authentication - proceed to /auth/ link firstly, and get the Token.
** Changing of the content available for author and admin only. Post owner able to delete comments related to his/her post.
